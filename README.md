# student-record-management
"Web-based Student Management System developed with HTML, CSS, JavaScript, PHP, and MySQL for managing student information."

## Project Description
The Student Management System is a web-based application developed to manage student records in a school. The system allows users to store, view, update, and delete student information easily.

This project demonstrates the use of web technologies and database integration for managing student data efficiently.

## Features
- Add new student records
- View student information
- Update student details
- Delete student records
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP (Local Server)

## Tools
- Visual Studio Code
- XAMPP
- phpMyAdmin
